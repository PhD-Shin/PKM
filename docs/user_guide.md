# Didymos 플러그인 100% 활용 가이드

"뭔가 쓰기 어렵다"고 느끼셨나요? Didymos는 복잡한 설정 없이 **"그냥 쓰면 구조가 생기는"** 것을 목표로 합니다. 딱 3가지 패턴만 기억하세요.

---

## 🚀 1. "목표"와 "프로젝트" 중심으로 글쓰기

AI가 내 노트를 8가지 타입(Goal, Project, Task 등)으로 분류합니다. 노트 제목이나 내용에 힌트를 주면 더 정확해집니다.

*   **Goal (목표)**: `# Goal: 박사 졸업`, `Objective: 스타트업 런칭`
*   **Project (프로젝트)**: `# Project: 3장 작성`, `MVP 개발 프로젝트`
*   **Task (할일)**: `- [ ] 문헌 조사하기`, `- [ ] API 명세서 작성`
*   **Insight (발견)**: `# Insight: RAG는 환각을 줄인다`

**팁**: 그냥 평소처럼 쓰세요. 시스템이 문맥을 분석해 "이건 프로젝트군", "이건 아이디어군"하고 분류해줍니다.

---

## 🗺️ 2. "지식 지도"로 내비게이션 하기

왼쪽 리본의 **Didymos 아이콘**을 클릭하세요.

1.  **빨간색(Goal) 노드**를 찾으세요. 내 삶의 가장 큰 목표들입니다.
2.  거기서 뻗어나가는 **파란색(Project) 노드**를 보세요. 목표를 달성하기 위한 구체적 프로젝트들입니다.
3.  **화살표(REQUIRES, ACHIEVED_BY)**를 따라가세요. "아, 이 목표를 위해 지금 이 프로젝트가 필요하구나"를 시각적으로 알 수 있습니다.

**활용법**:
*   "내가 지금 뭘 놓치고 있지?" → **고립된(연결 없는) 노드**를 찾아보세요.
*   "이 다음엔 뭘 해야 하지?" → **Goal에서 연결된 Project**를 따라가세요.

---

## 👆 3. "클릭"해서 깊게 파고들기 (New!)

그래프는 보기만 하는 게 아닙니다.

1.  관심 있는 **엔티티(점)를 클릭**하세요.
2.  상세 패널이 뜹니다:
    *   **요약**: 이 개념이 뭔지 AI가 3줄 요약해줍니다.
    *   **연결된 노트**: 이 개념이 등장하는 내 노트 목록입니다.
3.  **노트 제목을 클릭**하세요. 바로 그 글로 이동해서 수정하거나 내용을 추가할 수 있습니다.

---

## 💡 추천 워크플로우 (하루 루틴)

1.  **아침**: Didymos 그래프를 엽니다. 빨간색(Goal)을 보고 오늘 집중할 파란색(Project)을 확인합니다.
2.  **낮**: 노트를 막 씁니다. 회의록, 아이디어, 할 일... 형식 상관없이 씁니다.
3.  **저녁**: `Sync` 버튼을 누릅니다. AI가 내 잡동사니 글들을 "구조화된 지식"으로 바꿔 그래프에 업데이트해줍니다.

**핵심**: "정리는 AI가, 나는 글쓰기만." 이것이 Didymos의 철학입니다.
